Домашнее задание 22	Реализовать абстрактный класс для работы с таблицами БД
Реализовать абстрактный класс (модель) для работы с таблицами MySQL, а конкретно следующие методы:

find($id) - статический метод, поиск записи по идентификатору
findAll() - статический метод, выборка всех записей из таблицы
delete() - удаление записи
*** save() - сохранение
** Вместо save можно попробовать реализовать методы create и update
